Pillowtech code generator is used as a low cost water management tool. It is used to generate a downloadable arduino file where the user programs their arduino board and it becomes plug-and-play with various sensors.

Our goal is to focus on developing and fostering the adoption of Do-It-Yourself (DIY), low-cost, open-source technologies to improve farm-scale water management.

Made with: Python, JavaScript, HTML/CSS

**FOR DEVELOPER**

Have the master spreadsheet in the same directory as the rest of the files. In the terminal

Using the command: python3 build.py

Will fix up everything that was changed in the master spreadsheet up to that that point into the source code.

Using the command: python3 watch.py

Will listen in on all instances of the master spreadsheet being saved and will fix up the source code. Use CTRL+C to stop this
